## 📖 How to Run My Portfolio Locally

Want to check out my portfolio on your machine? Here's what you need:

### Prerequisites
- Python 3.8+
- pip or conda

### Installation

```bash
# Clone my repository
git clone https://github.com/Yogeswarachary/Portfolio.git
cd Portfolio

# Install dependencies
pip install -r requirements.txt

# Run the app
streamlit run app.py
```

The app will open at `http://localhost:8501`
